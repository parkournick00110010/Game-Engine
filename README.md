# My First Game engine with C++ and ECS Design

## dependencies
### Arch based distros
- base-devel
- [sdl2](https://archlinux.org/packages/extra/x86_64/sdl2/)
- lua5.3

### Debian based distros ( Like ubuntu )
- build-essential
- libsdl2-dev
- libsdl2-image-dev
- libsdl2-ttf-dev
- libsdl2-mixer-dev
- liblua5.3-dev
